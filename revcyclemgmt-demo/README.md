# RevCycleMGMT Training Demo

Ticket 1 scaffold for the RevCycleMGMT RCM training demo.

Canonical path:

```text
/mnt/z/BUSINESSES/RevCycleMGMT/06_PORTFOLIO_PROJECTS/revcyclemgmt-app-shell/revcyclemgmt-demo
```

## Stack

- Next.js 14 App Router
- TypeScript
- Tailwind CSS
- shadcn/ui
- Framer Motion
- `lucide-react`
- `@react-pdf/renderer`
- Inter and JetBrains Mono via `next/font/google`

## Commands

```bash
npm run dev
npm run build
npm run lint
```

Development URL:

```text
http://localhost:3000
```
