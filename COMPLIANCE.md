# Compliance Boundary

This repository is a synthetic product-console proof. It is not a production application, compliance certification, legal opinion, coding authority, payment guarantee, or clearinghouse certification.

The included intake values, readiness score, checklist, claim journey rows, payer mix, claim volume, dashboard metrics, and generated artifacts are synthetic proof data. They are not client extracts, payer reports, production remits, production claims, source-system exports, or patient records.

Public/demo mode allows only:

- Business-level workflow facts.
- Synthetic claim journeys.
- Synthetic dashboard metrics.
- No-PHI readiness planning.
- Public proof artifacts.

Production work requires:

- Formal agreements where appropriate.
- Minimum-necessary data handling.
- Role-based access controls.
- Audit logging and retention policies.
- Source-system validation.
- Secure file intake or controlled integration paths.
- Stakeholder review before any live metric is used for financial or operational decisions.
